# Project Aegis: 2026 Development Roadmap

This roadmap outlines the milestones for transitioning from a "Sovereign Blueprint" to a "Functional Digital Bodyguard."

<br>

## **Phase 1: The Iron & The Ink (Months 1-2)**
* **Hardware Procurement:** Acquisition of the Dual-RTX 5090 array and the Starlink Mini.
* **OS Hardening:** Baseline installation of a hardened Linux kernel on the Shield and Vault.
* **Physical Integration:** Custom ruggedization of the Pelican 1615 Vault.

<br>

## **Phase 2: The Core Intelligence (Months 2-3)**
* **Model Orchestration:** Deployment of Llama-3 (70B) and Mistral-Large-Sovereign using local quantization (GGUF/EXL2).
* **VRAM Optimization:** Benchmarking inference speeds across the GDDR7 bus to ensure real-time "advisor" capabilities.
* **MCP Integration:** Establishing the Model Context Protocol bridge between the Shield and the Vault.

<br>

## **Phase 3: The Perimeter (Months 4-5)**
* **Telemetry Scrubbing:** Live testing of the `gateway-policy.yaml` to ensure zero-leak status during Starlink operations.
* **The Tunnel:** Stress-testing the WireGuard/Tailscale mesh for high-latency "Industrial-Edge" environments.
* **Hardware MFA:** Implementation of FIDO2/YubiKey physical authentication for all remote requests.

<br>

## **Phase 4: The Harmony Protocol (Month 6+)**
* **The Steward Alpha:** Ingesting personal data into a local, encrypted vector database (RAG) for personalized advising.
* **Agentic Workflows:** Developing "Bodyguard" scripts that monitor personal data for intrusion or unauthorized access.
* **Field Testing:** Full-scale off-grid operation via EcoFlow power in a central New York environment.

<br>

## **Phase 5: Sovereign Systems & Professional Deployment (Month 12+)**

* **Aegis-Certified Hardware:** Selling pre-built, plug-and-play "Vault" units (from the $5k "Common Person" build to the $70k "Industrial" build). You aren't charging for the AI; you're charging for the labor, testing, and the Pelican-case integration.
* **Tiered Support Contracts:** Providing "Digital Bodyguard" monitoring services. If a user’s Shield detects a breach or a Starlink hardware failure, you (or your future team) provide the technical rescue. 
* **The "Steward" Pro-Distro:** While the core OS is free, you offer a "Professional Edition" that includes pre-configured vector databases for specific industries (e.g., local farming, independent journalism, or off-grid engineering).
* **Workshops & Installation:** Physical site-visits to set up "Community Vaults" for neighborhoods or small businesses, charging a premium for the "Human-in-the-loop" expertise.
